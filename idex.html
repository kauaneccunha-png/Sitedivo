Guia Rápido de Identificação de Cabos e Layouts de Rede

Quem é quem nos Meios Físicos? (Cabos)

Par Trançado

Clique para ver a função

O que é: Cabo de rede comum (Ethernet/Profinet).

Para que serve: Usado para ligar o CLP (Controlador Lógico Programável) ao computador do supervisor ou IHM. É o padrão para comunicação direta e rápida em curtas distâncias.

Fibra Óptica

Clique para ver a função

O que é: O famoso "cabo de vidro".

Para que serve: Usado para levar dados a longas distâncias na fábrica sem sofrer com a interferência magnética dos motores elétricos pesados. Se tem ruído elétrico, use este!

Wireless

Clique para ver a função

O que é: Rede sem fio (Rádio/Wi-Fi Industrial).

Para que serve: Usado para monitorar sensores que ficam em locais móveis (como pontes rolantes, AGVs) ou de difícil acesso onde passar cabo é inviável.
